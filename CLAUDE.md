# tech_prep — Quant Researcher Interview Prep

This vault is Bill's personal knowledge base for preparing for quant researcher interviews. Notes cover core quantitative topics tested in quant research roles (statistics, probability, time series, ML, linear algebra).

## Folder Structure

| Folder | Contents |
|--------|----------|
| `Probability/` | Probability theory, distributions, combinatorics, Bayes' theorem, common interview problems and formulas |
| `Statistics and Time Series/` | Statistical inference (hypothesis testing, MLE, confidence intervals), time series models (ARIMA, GARCH, stationarity, cointegration) |
| `Regression and Machine Learning/` | Linear/logistic regression, LASSO, Ridge, PCA, Fama-MacBeth, factor models, ML/deep learning concepts |
| `Linear Algebra/` | Matrix operations, eigenvalues/eigenvectors, decompositions (SVD, Cholesky), projections |

## Note Conventions

- Use `.md` files for all notes (Obsidian-compatible)
- Name files descriptively, e.g., `central_limit_theorem.md`, `kalman_filter.md`
- Add a `#tags` line at the top of each note for Obsidian linking
- Use `[[wikilinks]]` to cross-reference related notes

## Adding New Topics

If a topic doesn't fit the four folders above, create a new top-level folder and update this file with its description.
