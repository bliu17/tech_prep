## 1. 定理内容

  

把多元回归的自变量分成两块 $X_1$(要控制 / 剔除的)和 $X_2$(关心的):

  

$$y = X_1\beta_1 + X_2\beta_2 + \varepsilon$$

  

FWL 定理:$\hat\beta_2$ 可由以下三步得到,**与直接跑完整回归的 $\hat\beta_2$ 逐字相等**:

  

1. 把 $y$ 对 $X_1$ 回归,取残差 $\tilde y = M_1 y$

2. 把 $X_2$ 对 $X_1$ 回归,取残差 $\tilde X_2 = M_1 X_2$

3. 把 $\tilde y$ 对 $\tilde X_2$ 回归 → 系数即 $\hat\beta_2$,且**残差也与完整回归相同**


---

  

## 2. 关键性质:单侧 vs 两侧

  

**只中性化因子侧**(用 $\tilde X_2$ 配**原始** $y$)就足以恢复正确的**系数**:

  

$$\hat\beta_2 = \frac{\tilde X_2'\, y}{\tilde X_2'\,\tilde X_2} = \frac{\tilde X_2'\,\tilde y}{\tilde X_2'\,\tilde X_2}$$

  

(成立的原因:$\tilde X_2 \perp X_1$,而 $y$ 中能被 $X_1$ 解释的部分落在 $X_1$ 张成的空间里,自动与 $\tilde X_2$ 正交,贡献为零。)

  

| | 系数 $\hat\beta_2$ | 标准误 / t / $R^2$ / 残差 |

|---|---|---|

| 单侧($\tilde X_2$ 配原始 $y$) | ✅ 正确 | ❌ 错误 |

| 两侧($\tilde X_2$ 配 $\tilde y$) | ✅ 正确 | ✅ 正确 |

  

> **实务含义**:用 $\tilde f$ 排序选股、看因子边际预测力 → 单侧够用。

> 要做显著性检验 / 归因 → 必须两侧中性化,或直接跑多元回归。

  

---

  

## 3. 直觉:FWL = 遗漏变量无偏误的特例

  

单独跑 $y \sim \tilde X_2$ 确实遗漏了 $X_1$。OVB 公式:

  

$$\text{偏误} = \beta_1 \times (X_1 \text{ 对 } \tilde X_2 \text{ 回归的系数})$$

  

但 $\tilde X_2$ 按定义与 $X_1$ 正交,该辅助系数 = 0,故偏误 = 0。

**被省掉的变量与留下的变量不相关,省掉它就不污染斜率。**

## 4. 总结

1. FWL系数相等是无条件的;**标准误 / $R^2$ / 残差相等需两侧中性化**。

2. **普通 OLS**;带 L1/L2 惩罚的正则化(ridge/lasso)下此等式**不成立**。
